# Axionax Marketplace

Decentralized marketplace for compute resources on the Axionax network.

## Features

- Browse and purchase compute power
- List your own compute resources
- Smart contract escrow
- Token payments (AXX)

## Tech Stack

- Vite + React + TypeScript
- @axionax/sdk for blockchain interaction
- Ethers.js / Viem
- TailwindCSS

## Quick Start

```bash
npm install
npm run dev
```

## Development

```bash
npm run build    # Production build
npm run preview  # Preview production build
npm run lint     # ESLint check
```
